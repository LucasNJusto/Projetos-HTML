Elementos e Atributos

Elementos

Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final.

Exemplos: H1 até H6 p br html body

Elementos HTML vazios

Elementos HTML sem conteúdo são chamados de elementos vazios. br é um exemplo de elemento vazio.

Atributos HTML

Todos os elementos HTML podem ter Atributos Os atributos fornecem informações adicionais sobre os elementos São sempre especificados na tag inicial Os atributos geralmente vêm pares nome/valor como: nome="valor"

elemento a, com href de atributo vai especificar uma URL da página para qualquer link vai levar:

O atributo de estilo

O atributo style é usado para adicionar estilos a um elemento, como cor, tamanho e muito mais.

Olha esse parágrafo.

Sugestão: sempre use atributos em minúsculas recomendações da W3C O padrão HTML não exige nomes de atributos em letras minusculas.

Resumo

  1. Todos os elementos HTML podem ter atributos
  2. O href atributo de < a > especifica o URL da página para a qual o link vai
  3. O scr atributo de < img > especifica o caminho para a imagem a ser exibida
  4. Os atributos width e height, fornecem informações de tamanho para imagens
  5. O atributo alt, fornece um texto alternativo para uma imagem.
  6. O atributo Style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.
  7. O lang atributo da tag < html > declara o idioma da página da WEB
  8. O atributo title define algumas informações extras sobre algum elemento.
